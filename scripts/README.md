# Agentic OS scripts

Repo-local tooling for t8 Agentic OS. Run from the **repository root** with **Python 3.10+** (standard library only).

| Script | Purpose | Writes |
| --- | --- | --- |
| `_common.py` | Shared config/path helpers (imported by other scripts) | nothing |
| `graph_sync.py` | Validate graph JSON, sample-check paths, record freshness | Managed block in `.agentic/GRAPH_INDEX.md` |
| `route_task.py` | Route a task string to a context bundle | stdout JSON + `.agentic/CONTEXT/last_context.json` |
| `validate_memory.py` | Structural checks on memory, config, and graph | nothing (read-only) |
| `update_memory.py` | Sync graph metadata, detect git changes, refresh managed memory sections | `.agentic/` managed regions only |

## Usage

```bash
python scripts/graph_sync.py
python scripts/route_task.py "fix offline photo queue retry"
python scripts/route_task.py "src/lib/offline/photo-queue.ts" --explain
python scripts/validate_memory.py
python scripts/update_memory.py
python -m unittest discover -s tests
```
